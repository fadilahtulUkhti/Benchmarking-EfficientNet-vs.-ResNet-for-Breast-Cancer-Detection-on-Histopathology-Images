Histopathology imaging serves as a biomedical standard for micro-level disease detection, enabling Deep Learning model implementation to provide precise information and assist doctors. However, manual examination is time-consuming and susceptible to inter-observer variability. Recent advances in deep learning have demonstrated significant potential for automating medical image classification tasks. This study presents a comparative evaluation of two widely adopted convolutional neural network (CNN) architectures, namely ResNet50 and EfficientNetB2, for breast cancer detection using histopathological images. The models were trained and evaluated on the BreakHis dataset containing benign and malignant breast tissue samples acquired at different magnification levels. A comprehensive preprocessing pipeline, including image resizing, normalization, and data augmentation, was employed to improve model robustness. Performance was assessed using 5-fold cross-validation and standard evaluation metrics, including accuracy, precision, recall, F1-score, and Area Under the Receiver Operating Characteristic Curve (AUC). Experimental results demonstrate that both architectures achieved high classification performance, with ResNet50 obtaining an average validation accuracy of 90.35\% and an AUC of 0.9588. Meanwhile, EfficientNetB2 achieved competitive results with average validation accuracy 97.65\% and AUC of 0.9999 while requiring fewer parameters and computational resources. The comparative analysis highlights the trade-off between classification performance and model efficiency, providing valuable insights for the deployment of deep learning systems in computer-aided breast cancer diagnosis. The findings suggest that both architectures are suitable for histopathological image classification, with model selection depending on the target application and computational constraints.
